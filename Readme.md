### Flowerflow

### Requirements
1. php
2. webserver (apache)
3. composer

### Installation

### Step 1, install using composer

Install the module using composer.

```
composer require bthpan/flowerflow
```
Stand in the dir proj
```
Mkdir proj
Cd proj
```
rsync -av  vendor/bthpan/flowerflow/.  ./
